自動ログイン用Cookie（ltoken / ltuid）の取得手順と注意事項について

1

公式サイトへログイン

  • PCのブラウザで HoYoLAB を開く
  • 対象のアカウントでログインを完了させる
2

デベロッパーツールを起動

  • F12 キー、または右クリック ➔ 「検証」 を選択
  • 上部タブの 「Application（アプリケーション）」 をクリック
3

Cookieの値を抽出

  • 左サイドバーの「Storage」➔ 「Cookies」➔ 「hoyolab.com」を選択
  • フィルター欄に v2 と入力して絞り込む
  • ltoken_v2 と ltuid_v2 の「Value（値）」をコピー

注意事項

・共有禁止: この値はあなたのログインパスワードと同等の権限を持ちます、第三者に共有しないでください、本システム管理者はこれらの情報について、ログイン自動化のみに利用し、厳重に管理致します。
・ログアウト厳禁: ブラウザで「ログアウト」するとCookieは即座に無効化されます。
・有効期限: 数ヶ月で切れる場合があります。動かなくなったら再取得してください。

【管理者用】 GAS 貼り付け用テンプレート

const token = "ltoken_v2=取得した値; ltuid_v2=取得した値;";
